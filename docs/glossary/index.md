# Glossary

Definitions for **AI SDR**, **lead enrichment**, **programmatic SEO**, and related terms. Use this as a reference for top-of-funnel questions and to connect concepts to ZeroGTM’s [features]({{ site.baseurl }}/features/) and [use cases]({{ site.baseurl }}/use-cases/).

## Terms

| Term | Summary |
|------|---------|
| [What is an AI SDR?]({{ site.baseurl }}/glossary/ai-sdr/) | An autonomous agent that handles prospecting and qualification—distinct from a chatbot. ROI and cost vs. human SDRs. |
| [What is lead enrichment?]({{ site.baseurl }}/glossary/lead-enrichment/) | Appending data (firmographics, emails) to incomplete lead records. Includes waterfall enrichment. |
| [What is programmatic SEO?]({{ site.baseurl }}/glossary/programmatic-seo/) | Creating pages at scale using data and templates. Examples: integration pages, vertical landing pages. |
