FAQ

Common Questions

What is a "credit" exactly?
1 credit = 1 lead processed through 1 pipeline step. So scraping 50 leads costs 50 credits, then finding emails for those 50 leads costs another 50. If you bring your own API key for a step, that step costs zero credits.
What does "Bring Your Own Keys" mean?
Each pipeline step uses an external API (RapidAPI, OpenAI, Anymail, etc). If you plug in your own API key for a service, that pipeline step is free — no credits deducted. You only pay the API provider directly, usually much cheaper at scale.
Can I really self-host this for free?
Yes. ZeroGTM is MIT licensed. Clone the repo, run docker-compose up -d, add your API keys, and you have the full pipeline running on your infrastructure. No billing, no limits. The managed service is for people who don't want to maintain servers.
Do unused credits roll over?
Credits reset each billing period. This keeps pricing simple and predictable. If you consistently need more, upgrading your plan gives you a much better cost-per-credit rate.
What data sources do you support?
Currently: Google Maps (via RapidAPI) for initial lead sourcing, OpenWeb Ninja for contact enrichment, DataForSEO + AI for decision maker identification, and Anymail Finder for verified email discovery. More integrations are on the roadmap.
Is my lead data safe?
Every user's data is isolated at the database level via Row Level Security. Your leads, campaigns, and API keys are never visible to other users. Self-hosted users have full ownership. Managed users can export and delete all data at any time.
